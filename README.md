## UserExit - FormatDisk SSD.vbs
格式化所有硬盘，检测并选择 SSD。仅支持 Windows 10 PE 及以上。

## UserExit - SSD M2.vbs
检测并选择 SSD，优先选择 M2 SSD。
仅支持 Windows 10 PE 及以上。
